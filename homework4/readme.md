##Домашнее задание №4 (Регулярные выражения)

###Before

Создать в проекте следующую структуру папок "src/main/resources/" - это будет корневая директория для файлов программы, которые не являются исходным кодом.

Создать в resources файл regexp-basic.properties

Заполнить файл ответами на задания в формате:<br/> 
task1=ответ на задание 1<br/>
task2=ответ на задание 2<br/>
и т.д.<br/>

###Задание

Текст заданий в moodle

###Формат проверок 

1.Выполнить работу в проекте с домашним заданием на gitlab.com (разрешается и github.com).

2.Каждое домашнее задание должно быть выполнено в соответствующей отдельной ветке, например для текущей задачи homework4.<br/>
Рекомендуется создать ветку с пустой историей коммитов. Это можно сделать с помощью команды:
~~~
git checkout --orphan homework4
~~~

3.Готовую работу требуется отправить в систему автопроверки ДЗ: https://autoschool.rnd.lanit.ru/
